# Age Calculator 🧮

A web-based age calculator built using HTML, CSS and JavaScript.  
Made as part of my **CodeAlpha Web Development Internship** — May–August 2026 batch.

---

## 🔗 Live Preview
Open `index.html` in any browser — no installation needed.

---

## ✨ Features

- Enter your Date of Birth (Day / Month / Year)
- Calculates your exact age in **Years, Months and Days**
- Shows your **next birthday countdown**
- Displays the **day of the week** you were born
- Shows **total days lived**
- Full input validation (empty fields, invalid dates, future dates)
- Animated number counters
- Works on mobile and desktop
